# Top Camera Extension

상세 문서는 [top_camera_python/README.md](../top_camera_python/README.md) 참조.

## Quick Start

```bash
# Isaac Sim 실행 시
--ext-folder /path/to/isaac_sim_extensions --enable top.camera
```

## 주요 기능

- Global Camera: 단일 카메라로 전체 풀 촬영 (1920x1440)
- Per-Pool Camera: 개별 풀 카메라 선택적 퍼블리싱 (640x480)
- YOLO Dataset: Replicator 기반 bbox 데이터셋 수집
