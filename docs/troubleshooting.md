## 🚨 1. AttributeError: 'TopCamExtension' object has no attribute '_on_timeline_event'

### 🔴 문제 증상
* **에러 로그**:
  ```text
  AttributeError: 'TopCamExtension' object has no attribute '_on_timeline_event'
  At: /home/rokey/AquaSweep/isaac_sim_extensions/top_cam_ext/top_camera_python/extension.py(77): on_startup
  Failed to startup python extension.
  ```
* **영향**: 탑캠 익스텐션이 시동 도중 정지하여 시뮬레이션 카메라가 정상 부팅되지 않음.

### 🔍 원인 분석
* 개발 단계용 정보성 콘솔 `carb.log_warn` 메시지들을 청소하는 도중, 타임라인 전역 재생/정지 감지 콜백 함수 헤더인 `def _on_timeline_event(self, event):` 코드 라인 및 인덴테이션이 실수로 누락 및 소거되었습니다. 
* 이로 인해 `on_startup` 루틴에서 타임라인 이벤트 구독 등록 시 런타임 상에 해당 타깃 함수가 실존하지 않아 속성 누락 오류가 터졌습니다.

### 🟢 최종 해결 방법
* `extension.py` 내의 해당 메서드 정의부 헤더를 완벽히 재구현하고, 내부 Play/Stop 처리 구문의 클래스 수준 들여쓰기 정렬을 완벽히 복구하였습니다.
* **조치 후**: 핫 리로드 및 시동 에러가 영구히 해결되고 깨끗하게 카메라 연계 동작이 수행됨을 검증했습니다.

## 🚨 2. 수 수십 개의 유령 이물질이 검출되는 벽면 그림자 및 이미지 사각 경계 오탐지 현상

### 🔴 문제 증상
* 조명 평탄화 후 전체 반전 이진 마스크(`bitwise_not`)를 취했을 때, 수조 벽면의 두꺼운 그림자 테두리 및 이미지의 최외곽 1px 사각 경계선이 거대한 이물질 덩어리로 무더기 오탐지되어 이물질 수치가 수십 개로 비정상 폭증하는 현상.

### 🔍 원인 분석
* 조도 성분을 가우시안으로 나누는 조명 평탄화 과정에서 수조 바깥 음영 구역 및 벽면 최외곽은 배경 대조값이 낮아 반전 시 강한 흰색(255) 윤곽선으로 인식되었습니다.
* 이 경계선들이 모폴로지 팽창을 거치며 무더기로 쪼개져 수많은 가짜 이물질로 분할 팝업되었습니다.

### 🟢 최종 해결 방법
1. **면적 스펙트럼 필터 강제 적용**: 이물질의 유효 면적을 초소형 노이즈를 거른 `25px`부터, 수조 외곽선 전체가 잡히는 거대 덩어리를 소거한 `450px` 사이로 한정하는 듀얼 쓰레시홀드를 구현하여 거대 테두리가 이물질로 잡히는 현상을 수학적으로 원천 격파했습니다.
2. **원형 절삭 마스크 반경 조정**: 원형 수조 내부만 정밀 캡처하기 위해 적용하는 Mask의 반지름 오프셋 값을 기존 수조 타이트 핏에서 `-55px`로 넓혀 어두운 벽면 그림자를 마스크 바깥 영역으로 완벽하게 밀어내어 봉쇄했습니다.

## 🚨 3. 수면 부근의 흐릿하고 밝은 철갑상어(Sturgeon) 미탐지 및 조각 오탐지 현상

### 🔴 문제 증상
* 바닥 근처의 진하고 뚜렷한 상어들은 탐지 사각형(OBB)으로 잘 감지되나, 수면 위로 높이 헤엄쳐 올라와 중앙 조명 하이라이트 번짐 속으로 들어간 상어들이 배경 흰색과 뭉개지며 완전히 미탐지되거나 꼬리/머리 조각만 이물질로 쪼개져 인식되는 오작동 발생.

### 🔍 원인 분석
* 기존에 고정 임계값 이진화(`cv2.threshold` @ 220)를 쓰고 있었기 때문에, 조도 평탄화 이후에도 배경(255)과 명암 격차가 10~20 미만으로 흐릿하게 바랜 상어들의 픽셀 대조값(235~245 수준)이 220 기준선 이하로 전부 잘려 나가 배경으로 버려졌습니다.

### 🟢 최종 해결 방법
* **적응형 가우시안 반전 이진화(`cv2.adaptiveThreshold`)**로 전격 전환하였습니다.
* 고정 임계치 대신 국소 윈도우 격자(`BlockSize=151`) 내부의 가우시안 가중치 평균을 기준으로 상대적인 명암 차이를 추출함으로써, 흐릿한 상어의 연회색 몸통 전체 윤곽선을 아주 기민하게 복원하는 데 성공했습니다.
